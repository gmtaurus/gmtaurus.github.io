HTML 是什么？

HTML 指的是超文本标记语言 (Hyper Text Markup Language) HTML 不是一种编程语言，而是一种标记语言 (markup language)，用来描述网页的一种语言.

CSS 概述

CSS 指层叠样式表 (Cascading Style Sheets), 样式定义如何显示 HTML 元素, 样式通常存储在样式表中, 解决内容与表现分离的问题.
